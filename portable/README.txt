HOW TO RUN ON ANY COMPUTER:

WINDOWS:
  1. Install Python from python.org (tick "Add to PATH")
  2. Double-click: launch_windows.bat
  3. BrayoOS opens!

LINUX:
  1. Open terminal in this folder
  2. Run: bash launch_linux.sh
  3. BrayoOS opens!

MAC:
  1. Double-click: launch_mac.command
  2. BrayoOS opens!

REQUIREMENTS:
  - Python 3.8 or higher
  - tkinter (included with Python)
  - pip install httpx pillow
